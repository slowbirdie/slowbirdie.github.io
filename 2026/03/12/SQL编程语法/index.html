    5. 函数

    聚合函数

    函数 作用
    AVG(列) 平均值
    COUNT(列) 计数（可用 * 计行数）
    MAX(列) 最大值
    MIN(列) 最小值
    SUM(列) 求和
    FIRST(列) 第一个值 (MS Access)
    LAST(列) 最后一个值 (MS Access)

    字符串函数

    函数 作用 (示例)
    UCASE(列) 或 UPPER(列) 转大写
    LCASE(列) 或 LOWER(列) 转小写
    MID(列, 开始, 长度) 提取子串 (MySQL：SUBSTRING)
    LEN(列) 或 LENGTH(列) 字符串长度
    FORMAT(列, 格式) 格式化 (日期/数值，不同数据库语法不同)

    数值函数

    函数 作用
    ROUND(列, 小数位) 四舍五入

    日期函数

    函数 作用 (示例)
    NOW() 当前日期和时间 (MySQL)
    DATE() 提取日期部分
    其他数据库差异大，使用时查阅对应文档
